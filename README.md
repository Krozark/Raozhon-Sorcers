# Roazhon-Sorcers
Roazhon-Sorcers website

Install
=======

For developers
--------------

```
mkvirtualenv raozhon_sorcers --python=python3.6
pip install -r requirements.txt
gem install sass
```

For everyone
------------

```
pip install -r requirements.txt
```


Test
=====

```
python manage runserver
```
